# File Integrity Checker Using Python

## Overview

This project verifies file integrity using SHA-256 hashing. It detects whether a file has been modified by comparing the current hash with the original hash.

## Features

* SHA-256 Hash Generation
* File Integrity Verification
* Tampering Detection
* Security Monitoring

## Technologies Used

* Python
* Hashlib
